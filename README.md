<h1>Clg_Chat_Bot</h1>
<p>Developer: Pradhumn Pawar</p>
<p>I created a chat bot for my college using rule-based QnA.</p>

<h2>Project Overview</h2>
<p>Clg_Chat_Bot is a rule-based chatbot designed to assist students and staff at a college by providing quick answers to frequently asked questions. The bot is built using HTML, CSS, and JavaScript, offering a straightforward and efficient solution for campus-related queries.</p>

<h2>Features</h2>
<ul>
    <li>Rule-Based Q&A: Responds to predefined questions with accurate answers.</li>
    <li>User-Friendly Interface: Simple and intuitive design for easy interaction.</li>
    <li>Customizable: Easily extendable to include more questions and answers.</li>
</ul>

<h2>Technologies Used</h2>
<ul>
    <li>HTML: Structure of the chatbot interface.</li>
    <li>CSS: Styling and layout of the chatbot.</li>
    <li>JavaScript: Logic for handling user inputs and providing responses.</li>
</ul>

<h2>Project Structure</h2>
<p>The repository contains the following files:</p>
<ul>
    <li>index.html: The main HTML file containing the chatbot interface.</li>
    <li>style.css: CSS file for styling the chatbot.</li>
    <li>script.js: JavaScript file implementing the chatbot logic.</li>
</ul>
